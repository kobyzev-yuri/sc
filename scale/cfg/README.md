# Конфигурационные файлы Dashboard

Эта директория содержит конфигурационные файлы для dashboard, отделенные от кода для лучшей организации проекта.

## Файлы

- `feature_selection_config_relative.json` - Конфигурация признаков для относительных признаков
- `feature_selection_config_absolute.json` - Конфигурация признаков для абсолютных признаков
- `feature_selection_config.json` - Общая конфигурация (если используется)

## Назначение

Эти файлы хранят:
- Выбранные пользователем признаки для построения шкалы патологии
- Метаданные о методе подбора признаков
- Метрики качества модели
- Информацию об исходном эксперименте (если конфигурация загружена из эксперимента)

## Автоматическое создание

Директория `cfg/` создается автоматически при первом использовании dashboard, если её нет.

## Версионирование

Эти файлы могут быть добавлены в `.gitignore`, если вы хотите хранить их локально, или закоммичены, если они являются частью конфигурации проекта.
